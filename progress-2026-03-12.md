### Bugs fixed along the way

1. **Chicken-and-egg load deadlock** — TileGrid watched `[sourceBytes, pageRange]` together, so it returned early when `pageRange` was null and never emitted `loaded`, so `pageRange` never got set. Split into two watches: one for doc loading, one for range-dependent init.
2. **Grid not updating on column change** — `const reactive(...)` can't be reassigned; `v-model="settings"` silently failed. Fixed with `Object.assign(settings, $event)`.
3. **Flex layout didn't form a real grid** — rewrote tile container from flexbox+insert-col elements to `display: grid; grid-template-columns: repeat(columns, 1fr)`.
